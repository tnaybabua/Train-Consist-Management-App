# Train Consist Management App (UC3)

## Flow of Project

* Application starts
* Bogie IDs are added
* Duplicate IDs are inserted
* HashSet removes duplicates automatically
* Unique bogie IDs are displayed

## Objective

To ensure uniqueness of bogie IDs using HashSet.

## Topics Covered

* Set Interface
* HashSet
* add() method
* Uniqueness enforcement
* No duplicate data
